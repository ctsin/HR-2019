集团介绍
成立于 2003年10月 ，中国领先的小微金融服务企业，致力于通过微贷技术服务协助银行等金融机构为小微企业主、个体商户及工薪阶层提供快速简便、免抵押、免担保的小额贷款服务。
经过十四年的发展，中安信业已成长为一家多元化的金融集团公司。

邱春伟
南山分行业务主管 2014年3月入职
大学专业是会计学，择业时考虑到个人兴趣与发展，选择加入中安成为一名信贷员。入职三年多以来，自己的收入与收获都是成正比例增长。和自己的大学同学相比，他们大部分还是做着原来的会计岗位，收入增幅不大，而我在公司公平的薪酬体系下，通过自己的努力，月收入也从原来的4位数提高到5位数，消费水平与交际圈也提升了一个档次。
